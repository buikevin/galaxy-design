Accordion

A vertically stacked set of interactive headings that each reveal a section of content.

Single Selection

Is it accessible? Yes. It adheres to the WAI-ARIA design pattern. Is it styled? Yes. It comes with default styles that matches the other components' aesthetic. Is it animated? Yes. It's animated by default, but you can disable it if you prefer.

Multiple Selection

Can I open multiple items? Yes! This accordion allows multiple items to be open at the same time. How does it work? Set the type to "multiple" and you can toggle multiple items independently. Is this useful? Absolutely! It's great for FAQ sections or any content that users might want to compare.
